## Eclipse 設定 UTF-8
<!-- 簡介 -->
<!--more-->


## 打開設定畫面

Window -->> Preferences -->> Workspace 

## 設定UTF-8

Text file encoding 選擇Other: UTF-8 
New text file delimiter 選擇Other: Unix
設定完後點選 Apply and Close
