# Eclipse 用 Open Type Hierarchy 查看類的繼承關係
<!--more-->

用Open Type Hierarchy可以查看類的繼承關係，可以在Hierarchy窗口看到繼承層次的導航。  

在方法或類名上點擊右鍵，選擇Open Type Hierarchy即可，快捷鍵是F4。
