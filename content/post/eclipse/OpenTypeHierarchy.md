## Eclipse 用 Open Type Hierarchy 查看類的繼承關係
<!--more-->

## 前言

有時會想要知道繼承類別的關係，此時可以使用ide 內建的功能來解決

## 使用 Eclispe

用Open Type Hierarchy可以查看類的繼承關係，可以在**Hierarchy**窗口看到繼承層次的導航。  

在方法或類名上點擊右鍵，選擇**Open Type Hierarchy**即可，快捷鍵是**F4**。
