## **從 0 開始的 JAVA 生活 No.1 Java 第一支 java 程式**

<!--more-->

### **建立一個HelloJava.java 的檔案**

```java
/**
 * HelloJava
 */
public class HelloJava {

	public static void main(String[] args) {
		System.out.println("Hello java");
	}

}
```

### **編譯程式**

     **語法 ： javac xxxx.java**

EX: 

```bash
javac HelloJava.java
```

### **執行程式**

    **語法 ： java xxxx**

EX :

```bash
java HelloJava
```
